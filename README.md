# Mubsir 🧭

**Mubsir** is a clean, modern web application designed to act as a guidance compass for high school graduates and students exploring various university majors. It provides clear insights into different technical and engineering disciplines, their key fields of study, potential career pathways, and general benefits.

## 🎨 Theme & UI Update
The user interface has been recently upgraded to a **Modern Tech Palette (Deep Navy & Soft Electric Blue)** for optimal readability, smooth sidebar navigation, responsive card structures, and a consistent UI hierarchy across all branch files.

## 📁 Project Architecture
The workspace contains the following streamlined files:
* `index.html` - Main landing page with comprehensive domain paths.
* `Computer-Majors.html` - Overview of computing disciplines.
* `Engineering-majors.html` - Overview of engineering tracks.
* `Information-Systems.html` - Information Systems specific focus.
* `Information-Technology.html` - Detailed IT major focus.
* `Computer-Science.html` - Computer Science pathway specifications.
* `CyberSecurity.html` - Cybersecurity field analysis.
* `AerospaceEngineering.html` - Aerospace pathway specifications.
* `CivilEngineering.html` - Civil major parameters.
* `ElectricalEngineering.html` - Electrical tracking details.
* `Industrial-Engineering.html` - Industrial structures mapping.
* `MechanicalEngineering.html` - Mechanical track scope.
* `Mining-Engineering.html` - Mining focus breakdown.
* `NuclearEngineering.html` - Nuclear systems specialization layout.
* `styles.css` - Global streamlined responsive stylesheet.
* `script.js` - Safe, non-blocking sidebar toggle and core DOM interactions.
